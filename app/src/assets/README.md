# FishingInsights Assets

## Image Attribution

All images used in this application are from Unsplash and are free to use under the Unsplash License.

### Required Images

Please download the following images from Unsplash and save them in this directory:

1. **hero-ocean.jpg** (1920x1080 recommended)
   - Search: "ocean waves fishing" or "calm ocean water"
   - Suggested: https://unsplash.com/s/photos/ocean-waves
   - Use for: Home page hero background

2. **section-header-ocean.jpg** (800x200 recommended)
   - Search: "ocean texture" or "water surface"
   - Suggested: https://unsplash.com/s/photos/ocean-texture
   - Use for: Section header backgrounds (subtle)

3. **empty-state-fishing.jpg** (600x400 recommended)
   - Search: "fishing rod empty" or "fishing equipment"
   - Suggested: https://unsplash.com/s/photos/fishing-rod
   - Use for: Empty states (favourites, locations)

### Attribution Template

When downloading images, note:
- Photographer name

- Image URL

These will be displayed in the References page attribution section.
